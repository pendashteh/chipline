# chipline
Console for ChatGPT

## Installation
Place your API key at the root directory and call it `key`:
```
$ cat ./key
sk-I0j2C2Co...SjCz45
```
Also, make sure `console` is executable.
```
$ chmod +x ./console
```

## Run
```
$ cd /path/to/chipline
$ ./console
> 
```
Now you can prompt via the console.
